## Example Project

1.) Clone this repository
2.)	add this line to your Podfile:
pod 'VWWClusteredMapView', :git => 'https://github.com/zakkhoyt/VWWClusteredMapView', :branch => 'cocoapods'

3.) Run pod install
4.) Open VWWClusteredMapViewExample.xcworkspace


## Requirements

- iOS 8.0 or higher
